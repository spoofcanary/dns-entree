# Code of Conduct

This project adopts the Contributor Covenant, version 2.1.

The full text is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Reporting

Report violations privately to: conduct@spoofcanary.com

Reports are handled confidentially. The project maintainers will respond
within 5 business days.

## Scope

This Code of Conduct applies in all project spaces (repository, issues,
pull requests, discussions) and when an individual is officially
representing the project in public spaces.
